# University of Haripur GPA Management System

LIVE AT: [https://hamidshafiq1.github.io/scientific-calculator/ ](https://hamidshafiq1.github.io/GPA-Calculator/)

Repository contains the GPA calculation logic and implementation for the University of Haripur.

## Included Files
* `index.html`: Web-based graphical interface for GPA calculation[cite: 1].
* `main.cpp`: C++ console application for GPA management[cite: 2].

## Usage
* **Web UI:** Open `index.html` in any modern web browser to utilize the graphical interface.
* **Console Application:** Compile `main.cpp` using a standard C++ compiler (`g++ main.cpp -o gpa_calc`) and execute the resulting binary.
